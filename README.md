# Inventaris Barang

Merupakan aplikasi untuk memudahkan dalam melakukan jual beli barang sekaligus mengelola data barang. Aplikasi ini memiliki beberapa level pengguna, yaitu admin, kasir, dan manager, yang masing-masing memiliki tugas dan hak akses berbeda.

Admin = Mengelola data barang, merek, dan distributor  
Manager = Mengelola pegawai, transaksi, serta melihat laporan seluruh barang dan laporan barang habis  
Kasir = Melakukan transaksi penjualan dan mencetak struk
